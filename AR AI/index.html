Course Query:

The course query questions must be related to a specific course. Not all relevent information may be available but you can ask for the school webpage for more Information or the link to the course page.

An example weould be "What are the admission requirements for Mathematics?"


Country Query:

The country query questions must be related to a specific Country. Each country has specific support available so if you dont know what to look for you can ask for a link to the page for more detail

An example would be "What Visa information should I know when coming from Australia?"


Website Query:

Website queries can asnwer questions from the tcd.ie website, this being the study and academic registry sections. You need to be pretty specific to get an accurate result because there is a lot of information available. This is alos the slowest searach for the same reason.

An example would be "How to register for garda vetting?"
